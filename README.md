# Student Performance Prediction System

## Project Overview

This project predicts student academic performance using Machine Learning algorithms. The system analyzes factors such as study hours, attendance, internal marks, and assignment scores to determine whether a student is likely to pass or fail.

The project helps educational institutions identify at-risk students and improve academic performance through data-driven decision making.

---

## Objectives

* Predict student performance using machine learning
* Compare multiple machine learning algorithms
* Identify students who may be academically at risk
* Visualize student performance trends
* Improve educational decision making

## Dataset Description

The dataset contains academic details of students including:


Study_Hours     - Number of hours studied daily 
Attendance      - Attendance percentage         
Internal_Test_1 - Internal test 1 marks         
Internal_Test_2 - Internal test 2 marks         
Assignments     - Assignment scores             
Final_Marks     - Final exam marks              
Final_Result    - Pass/Fail target variable     

---

## Methodology

### Step 1: Data Collection

Collected student academic dataset in CSV format containing study hours, attendance, marks, and assignment details.

### Step 2: Data Preprocessing

* Removed missing values
* Renamed columns
* Created target variable (Pass/Fail)
* Prepared data for machine learning models

### Step 3: Exploratory Data Analysis

Performed visualization and analysis of:

* Study hours vs marks
* Attendance distribution
* Pass vs fail count
* Feature correlation

### Step 4: Feature Selection

Selected important academic features:

* Study Hours
* Attendance
* Internal Marks
* Assignment Scores

### Step 5: Model Selection

Applied multiple machine learning algorithms:

* Logistic Regression
* Decision Tree
* Naive Bayes
* Support Vector Machine (SVM)

### Step 6: Model Training

* Split dataset into training and testing data
* Trained all models using training data

### Step 7: Prediction

Implemented manual input functionality to predict whether a student will pass or fail.

### Step 8: Evaluation

Evaluated model performance using:

* Accuracy
* Precision
* Recall
* F1-score
* Confusion Matrix
